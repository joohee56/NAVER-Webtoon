## 소개
* **제작 목적** : 실제 서비스 되고 있는 서비스를 제작해봄으로써 실사용자가 필요로 하는 기능을 인지하고 배운 기술을 적용하기 위함
* **역할** : 개인 프로젝트
* **제작 기간** : `2024.02.19 ~ 2024.05.29`
</br>

## 사용자 주요 유스케이스
1. 웹툰 관람
2. 작가의 웹툰 등록
3. 검색
