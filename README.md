## 소개
* **제작 목적** : 실제 서비스 되고 있는 서비스를 제작해봄으로써 실사용자가 필요로 하는 기능을 인지하고, 학습한 기술을 적용해보자.
* **역할** : 개인 프로젝트
* **제작 기간** : `2024.02.19 ~ 2024.05.29`
</br>
